# Task Manager Application

A full-stack task management web application built using Node.js, Express, MySQL, HTML, CSS, and JavaScript.

## Features

- User Registration
- User Login
- Password Hashing using bcrypt
- Session Management
- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- Responsive Design

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Packages
- bcrypt
- express-session
- mysql2

### Tools
- VS Code
- Postman
- Git
- GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/Astro-Gojo/task-manager-app.git
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```

Open:

```
http://localhost:5000
```
